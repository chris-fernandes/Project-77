My First Console Project

Visual Studio Code is platform for coding in a number of languages. Download VS Code for your particular softare and open any folder you want to work in. Install live server to check the output of your code.

Console can be used to check errors as well as display anything to check a certain portion of the code. It's used as a check point where you can check parts of your code. To open console right click on the web browser and inspect go to console option and here's the console open for you!!! To print anything in console write console.log() and write your message/ anything you want to print within the brackets.
